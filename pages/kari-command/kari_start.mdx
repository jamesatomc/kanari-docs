# Kari Start Command

The `kari start` command initializes a local Kanari node for development.

## Prerequisites

Before using `kari start`, ensure:
- Kanari CLI is installed
- A wallet is generated using `kari keytool generate`

## Usage

```bash
kari start [options]
```

## Common Errors

### No Wallet Found
If you see:
```bash
No wallet found!
Please create a wallet first using:
kari keytool generate
```

## Wallet Setup

1. Generate a new wallet:
```bash
kari keytool generate
```

2. When prompted, select mnemonic length:
```bash
Enter mnemonic length (12 or 24):
24
```

## Usage

The `kari start` command initializes a local Kanari node for development.

## Network Configuration

When running `kari start`, you'll be prompted to configure your network:

```bash
kari start

Choose a network type:
1. devnet
2. testnet
3. mainnet
Enter your choice [1-3]:
```

Configuration options:
- Network type (1 devnet) Enter
- RPC port (default: 3031) Enter
- Network domain (default: devnet.kanari.network) Enter
