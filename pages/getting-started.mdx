# Getting Started

In this section, we will guide you through the initial steps required to set up your development environment for working with the Kanari Network.

## Prerequisites

Before you begin, ensure you have the following installed on your machine:

- Rust: The programming language used to build the Kanari Network. You can install Rust by following the instructions on the [official Rust website](https://www.rust-lang.org/tools/install).

## Installation Steps

1. **Install Rust**

   Open your terminal and run the following command to install Rust:

   ```sh
   curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
   ```

2. **Verify Installation**

    After the installation is complete, verify that Rust is installed correctly by running:

    ```sh
    rustc --version
    ```
